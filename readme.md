Website For Mum
========================

####A website created for Mum so that the link to her friends for them to wish her a Happy Birthday!!

#####Technologies Used:

  - Sinatra
  - Sinatra-partial
  - ERB
  - HTML5
  - JQuery
  - CSS3 
  - postgreSQL
  - RSpec
  - Capybara
  - Bootstrap
  - Data Mapper

How to Use:
------------

Clone the Repository:
```shell
$ git clone git@github.com:Scully87/Fam.git
```

Open the Directory:
```shell
$ cd Fam
```
Install the files:
```shell
$ bundle install
```

Open from terminal:
```shell
$ rackup
```

Url for Browser:
```
  http://localhost:9292/
```
